Lesson 1: Introduction to HTML

Learn the basics of HTML and how it is used to structure content on the web.

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page and consists of elements that define different parts of the content.

Basic HTML Tags

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <a>: Anchor (Links)
  • <img>: Images
  • <div>: Division

Example

                        
                            <!DOCTYPE html>
                            <html>
                            <head>
                                <title>My First HTML Page</title>
                            </head>
                            <body>
                                <h1>Welcome to HTML</h1>
                                <p>This is a paragraph.</p>
                                <a href="https://example.com">Visit Example</a>
                            </body>
                            </html>
